Reads the current session todo list. Returns all todo items with their status, priority, and content.

USE THIS TOOL FOR:
- Checking what tasks are pending at the start of a conversation
- Reviewing remaining work before starting a new task
- Confirming what was completed and what still needs to be done
- Recovering context after a gap in the conversation

WHEN TO CALL:
- At the start of a new session to see what's in progress
- Before starting a new task to prioritize correctly
- After completing a task to understand what comes next
- Whenever you're uncertain about what to do next
- After every few messages on a multi-step task to stay on track

USAGE NOTES:
- Takes NO parameters. Pass an empty input — do NOT include a key like `"input"` or `"empty"`.
- If no todos exist, returns an empty list.
- Use the returned list to plan next steps, not just to acknowledge it.
