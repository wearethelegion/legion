Searches the web in real time using Exa AI. Returns current information from live websites.

USE THIS TOOL FOR:
- Getting up-to-date information beyond your training knowledge cutoff
- Researching current events, recent releases, or live data
- Finding documentation, changelogs, or announcements for tools and libraries
- Any question where freshness matters

DO NOT USE THIS TOOL FOR:
- Searching LOCAL codebase files → use Grep or Glob instead
- Fetching a SPECIFIC known URL → use WebFetch instead
- Searching for code examples and API docs → use CodeSearch instead (higher quality for programming)

KEY PARAMETERS:
- query: Search query string (required). Be specific. Include the current year for time-sensitive searches.
- num_results: Number of results to return (configurable).
- live_crawl: `"fallback"` (use cached, fall back to live if unavailable) or `"preferred"` (prioritize live crawling).
- search_type: `"auto"` (balanced), `"fast"` (quick results), `"deep"` (comprehensive).

USAGE NOTES:
- The current year is {{year}}. Always use this year when searching for recent information.
  - CORRECT: search "TypeScript 5.8 release notes 2026"
  - WRONG: search "TypeScript latest release notes 2025" (wrong year)
- Results include content from the most relevant websites, not just links.
- Domain filtering and advanced search options are available if you need to restrict sources.
