Signals the end of the planning phase and prompts the user to switch to Build Agent mode to begin implementation.

USE THIS TOOL WHEN:
- You have written a complete, reviewed plan to the plan file
- All clarifying questions have been answered
- The plan is ready for implementation and you have nothing further to add

DO NOT USE THIS TOOL WHEN:
- The plan is not yet complete or still has open questions
- The user has indicated they want to continue planning or add more details
- You haven't written the plan to a file yet

BEHAVIOR:
- Asks the user whether to switch to Build Agent for implementation.
- The plan file remains available as context for the implementing agent.
- Only call this once the plan is truly complete and actionable.
