# Changelog

All notable changes to this project will be documented in this file.

## [1.1.64] - 2026-03-10

### Added
- First public release under `@wearethelegion/legion` on npm
- Multi-platform binaries: macOS (ARM64, Intel), Linux (x64, ARM64, musl), Windows
- GitHub Release with pre-built binaries
- CODE_OF_CONDUCT.md (Contributor Covenant v2.1)
- CONTRIBUTING.md with development setup guide
- PR template for consistent contributions
- Discord community server

### Changed
- Full rebrand from OpenCode to Legion (817 files, 22,400+ changes)
- All documentation updated with Legion branding and links
- Install URLs point to wearethelegion.com
- Discord links updated to Legion server

### Fixed
- Replaced all legacy domain references (legion.dev → wearethelegion.com)
- Restored `@gitlab/opencode-gitlab-auth` package reference (real npm dependency)
- Added missing `protobufjs` dev dependency
